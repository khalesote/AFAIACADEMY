Academia de Inmigrantes - Aplicación Móvil Completa

DESCRIPCIÓN:
Este archivo contiene el código fuente completo de la aplicación móvil 
"Academia de Inmigrantes", una plataforma educativa integral para el 
aprendizaje de español dirigida a inmigrantes.

La aplicación incluye los siguientes módulos y funcionalidades:

1. ESCUELA VIRTUAL
   - Sistema de aprendizaje de español (niveles A1, A2, B1, B2 según MCER)
   - Profesor Virtual con tutoría automatizada
   - Sistema de ejercicios interactivos
   - Sistema de evaluación y seguimiento de progreso
   - Componentes de clases de español
   - Sistema de diplomas y certificaciones

2. PREFORMACIÓN PROFESIONAL
   - Sistema de formación profesional para inmigrantes
   - Generador de CVs y plantillas profesionales
   - Sistema de pagos integrado
   - Agenda y gestión de cursos profesionales
   - Café Literario y recursos educativos
   - Libros descargables

3. CORE E INFRAESTRUCTURA
   - Arquitectura base de la aplicación
   - Sistema de autenticación y registro de usuarios
   - Sistema de pagos (Stripe y Cecabank)
   - Reconocimiento facial y seguridad
   - Configuración de la aplicación (Firebase, Expo, etc.)
   - Backend API y servicios

TECNOLOGÍAS UTILIZADAS:
- React Native / Expo
- TypeScript / JavaScript
- Node.js (Backend)
- Firebase
- Stripe (Pagos)
- Cecabank (Pagos)

PLATAFORMAS:
- Android
- iOS
