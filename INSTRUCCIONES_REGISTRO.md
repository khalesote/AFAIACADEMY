## Archivos Preparados

1. **1_EscuelaVirtual_20251202.zip** (1.83 MB)
2. **2_PreFormacion_20251202.zip** (0.06 MB)
3. **3_CoreInfraestructura_20251202.zip** (0.09 MB)

## Pasos para Completar el Registro

### 1. Completar las Portadas

Antes de subir los archivos, debe:

1. Abrir cada archivo `PORTADA_*.txt`
2. Reemplazar `[Indique aquí su nombre y apellidos completos]` con su nombre completo
3. Guardar los archivos

### 2. Agregar las Portadas a los ZIPs

Cada archivo ZIP debe incluir su portada correspondiente en la raíz:

- `PORTADA_1_EscuelaVirtual.txt` → dentro de `1_EscuelaVirtual_20251202.zip`
- `PORTADA_2_PreFormacion.txt` → dentro de `2_PreFormacion_20251202.zip`
- `PORTADA_3_CoreInfraestructura.txt` → dentro de `3_CoreInfraestructura_20251202.zip`

**Opción A - Manual:**
1. Abrir cada ZIP con WinRAR, 7-Zip o el explorador de Windows
2. Arrastrar el archivo PORTADA correspondiente dentro del ZIP
3. Guardar

**Opción B - Automático:**
Ejecutar el script `agregar_portadas.ps1` que se creará a continuación.

### 3. Subir los Archivos

1. Acceder al enlace proporcionado:
   https://sede.mcu.gob.es/rpi5/webpages/publicoSeguro/ejemplaresAccesoExt.seam

2. Cumplimentar el formulario:
   - **NIF/NIE**: [Su NIF o NIE]
   - **Número de expediente**: [El número que le proporcionaron]
   - **Ámbito o comunidad autónoma**: [La que seleccionó]

3. Subir cada uno de los 3 archivos ZIP (uno por uno)

### 4. Verificar Requisitos

✅ **Tamaño máximo**: Cada archivo es menor a 10 MB (requisito cumplido)
✅ **Formato**: ZIP (formato aceptado)
✅ **Portada**: Cada ZIP incluye portada con nombre del autor y título
✅ **Contenido**: Código fuente completo de cada módulo

### 5. Descargar Justificante

Después de subir cada archivo, descargue el justificante de entrega del ejemplar.
